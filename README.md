# Apple Services Manager v2.0

A Windows batch script utility to manage Apple services on your PC. This tool helps you easily start, stop, restart, and check the status of Apple-related services.

## Requirements

- **Windows Operating System**
- **Administrator Rights** - The script must be run as Administrator

## Features

The script provides a menu-driven interface with the following options:

| Option | Description |
|--------|-------------|
| 1 | **START Services** - Start all Apple services |
| 2 | **STOP Services** - Stop all Apple services |
| 3 | **CHECK Status** - View the current status of all Apple services |
| 4 | **RESTART Services** - Stop and start all Apple services |
| 5 | **EXIT** - Close the application |

## Managed Services

The following Apple services are managed by this script:

- Apple Mobile Device Service
- Bonjour Service
- iPod Service
- Apple Mobile Device Helper
- iCloud Services
- APSDaemon

## Usage

1. Download or clone this repository
2. Right-click on `apple_services.bat`
3. Select **"Run as Administrator"**
4. Use the menu to select your desired action (1-5)
